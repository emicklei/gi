## treerunner

Small program used for development to run example programs in folders.
It runs each of them and counts succcesses and failures.

## usage

    treerunner .

Perform a dry run using

    treerunner -dry .
    
### env

If the following environment variables are set, they will be used to control the behavior of the runner.

GI_IGNORE_EXIT
GI_IGNORE_PANIC
